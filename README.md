# FlappyBird With Neural Evolution of Augmenting Topologies (NEAT)

#### To run the game,

0. (Optional) Create a virtual environment
1. Install the requirements.txt
2. Run the game.py file
3. That's it.

### Tools

- Python 3.10
- Pygame
- NEAT-Python
